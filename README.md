# Powershell scripts

Just a bunch of useful powershell scripts

### 1. rename-video-as-per-exif.ps1

when you have recovered videos from currupt hdd etc. and the names are all some random string, this can be used to rename them into video-date-time-version.extension format using exif data


### 2. rename-img-as-per-exif.ps1

when you have recovered images from currupt hdd etc. and the names are all some random string, this can be used to rename them into image-date-time-version.extension format using exif data


### 3. move_files_to_date_folder.ps1

move files with a perticular filename prefix into folder


### 4. add_exif_data_to_img.ps1

we add custom exif data to the image file
